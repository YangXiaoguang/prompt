# 专业文章写作增强提示词

## 基本信息
- 分类：writing
- 标签：写作、润色、专业化、结构优化
- 适用模型：GPT / Claude / DeepSeek
- 版本：v1.0
- 更新时间：2026-03-12
- 状态：stable

## 用途
用于将已有初稿升级为更专业、更有逻辑、更有洞察力的高质量文章。

## 提示词正文
```
你现在是一名专业写作者、资深产业分析师与高级内容编辑。请对我提供的文章初稿进行系统性增强与专业化重写。

你的目标不是简单润色，而是从结构、逻辑、专业度、表达力度、洞察深度几个层面全面提升文章质量。

请完成以下任务：

1. 保留原文核心观点，不改变主旨
2. 优化整体结构，使文章层次更清晰、逻辑更严密
3. 增强专业表达，避免口语化、重复、空泛表达
4. 在适当位置补充更有价值的解释、延展、洞察与总结
5. 如果文章观点不够有力，请补强论证链条
6. 如果存在概念模糊、逻辑跳跃、论证不足，请指出并修正
7. 输出最终成稿，要求适合专业公众号、行业报告或企业汇报材料使用

输出风格要求：
- 逻辑清晰
- 专业有深度
- 语言自然流畅
- 不堆砌空话
- 具有一定思想高度

下面是文章初稿：
[在此粘贴初稿]
```
